### Hands on Learning on HTML and CSS via #My profile web-page

This is a **personal portfolio webpage** built using **HTML and CSS**. It showcases my skills, projects, and provides a way for people to contact me. The page is fully responsive and visually appealing with a modern theme.

---

## Features

- **About Me** section with profile picture and description
- **Skills** section with skill badges and progress indicators
- **Projects** section showcasing personal or professional projects
- **Contact** section with email, social links, and a clean form layout
- **Footer** with copyright and social links

---

## Technologies Used

- HTML5
- CSS3
- Git & GitHub for version control and hosting
